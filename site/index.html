Two layers. One honest answer.

Blockers say “ERR_PHISHING_DETECTED”. SafeSurf says “This page pretends to be your bank — close it.” Here is what happens under the hood:

Instant · offline · private

Local heuristics

  • Typosquatting — paypa1.com, look-alike foreign letters
  • Brand hidden in a fake address — paypal.com.evil.xyz
  • Scam script phrases: fake prizes, fake virus alerts, “call support”, gift-card payments, remote-access requests
  • Dark patterns: countdown timers, “do not close this window”
  • Card & password fields on unprotected connections
Claude AI · structured verdict

AI second opinion

  • Fires only when heuristics find something — normal browsing never leaves the machine
  • Claude weighs the page text, address and heuristic evidence
  • Returns a decisive verdict + a one-sentence explanation written for a 75-year-old
  • And one concrete action: “Close this page. Don’t enter your card.”
  • API unavailable? The heuristic warning stays. Fail-safe by design.

✅ Safe

No banner, no nagging. SafeSurf stays silent on normal pages.

⚠️ Be careful

Something feels off. Take your time, don’t enter personal details.

🛑 Danger

Strong scam signals. Big red banner, plain words, one clear action.

Install in one minute

SafeSurf is an open-source Chrome extension (Manifest V3, TypeScript, Claude API).

Clone the repo and run npm install && npm run build in extension/.
Open chrome://extensions, turn on Developer mode.
Click Load unpacked and pick the extension/dist folder.
Done — AI analysis works out of the box via SafeSurf Cloud. Open a demo above. (Power users can plug in their own Anthropic key.)
