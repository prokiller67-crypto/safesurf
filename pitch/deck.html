The solution

A guardian that speaks human.

SafeSurf watches every page and shows a big, high-contrast traffic-light banner the moment something is wrong — with one plain-language explanation and one concrete action.

  • 🛑 Danger / ⚠️ Caution / ✅ silence — no nagging on safe pages, loud when it matters
  • Huge type, high contrast — designed for older eyes
  • One action — “Close this page”, not a settings menu
  • Zero jargon — “fake website”, never “phishing domain”
  • Body font: Atkinson Hyperlegible — a typeface built for low-vision readers
  • Works without setup — heuristics protect even with no API key

How it works

Two layers. Fail-safe by design.

Layer 1 · instant · offline

Local heuristics

  • Typosquatting: paypa1.com, look-alike unicode
  • Brand-in-address tricks: paypal.com.evil.xyz
  • 12 scam-script patterns: fake prizes, fake virus alerts, gift-card payments, remote access
  • Dark patterns: countdowns, “do not close”
→
Layer 2 · Claude AI

AI second opinion

  • Fires only when Layer 1 flags something
  • Weighs URL + page text + heuristic evidence
  • Structured JSON verdict: risk, explanation, advice
  • Written for a senior, decisive by prompt design
→
Verdict

The banner

  • Heuristic warning appears in <50 ms
  • AI upgrades it seconds later
  • API down? Heuristic warning stays
  • Safe page? Total silence

Technical implementation

Small, honest, production-shaped.

  • Chrome Manifest V3 — TypeScript, esbuild, content script + service worker
  • Claude Opus 4.8 via official SDK with structured outputs (output_config.format) — the verdict is schema-validated JSON, never free text
  • Prompt-injection hardened — page text is fenced as untrusted content
  • Privacy-first — page text leaves the machine only when heuristics flag it; normal browsing is never sent anywhere
  • Key stays local — chrome.storage, user’s own browser profile
  • Tested — heuristic engine verified against scam & legit URL corpus
